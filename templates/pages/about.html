About Veloria

WordPress code search, reimagined

Veloria is a fast, full-text code search engine for the WordPress ecosystem. It indexes the source code of every plugin, theme and core version published on wordpress.org, letting developers search across the entire directory in seconds. Veloria is the successor to wpdirectory.net, rebuilt from the ground up with a modern stack and a focus on reliability.

Technology

Under the hood, Veloria relies on Google's codesearch library, which uses trigram indexing to perform regular-expression searches over large codebases at speeds, by only searching files that can potentially contain a match (using the trigram indexing).

Plugin and theme metadata is sourced from AspireCloud, FAIR's community-driven mirror of the wordpress.org API. AspireCloud provides a reliable, independent gateway to the WordPress directory data that Veloria depends on.

Sponsorship

Veloria is an open-source project maintained in spare time. If you find it useful and would like to help cover hosting and development costs, sponsorship is welcome via GitHub Sponsors. Every contribution, no matter how small, helps keep the service running.

Licenses

The Veloria source code is released under the MIT License. Third-party libraries used by the project retain their own respective licenses. WordPress plugin and theme source code displayed in search results remains subject to the license chosen by each individual author.

Privacy

Veloria collects the minimum data necessary to operate. Searches are stored so they can be shared and revisited; searches marked as private are visible only to the user who created them. If you sign in via a third-party provider, Veloria stores your display name and avatar URL solely to personalise your session. No analytics trackers or advertising scripts are loaded. Server logs are retained for a limited period for debugging and abuse prevention, then deleted.
